## ✨ Features

- **AI-generated facts** — Every fact is freshly generated by Claude (no stale static list)
- **6 topic categories** — Earth, Cats, Space, History, Science, Ocean
- **Zero dependencies** — Single HTML file, no npm, no bundler
- **Responsive design** — Works on desktop and mobile
- **Loading state** — Animated dots while fetching
- **Error handling** — Graceful fallback on API failures

## 🚀 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/Nepul1234/Fun-Facts-Generator-App.git
cd fun-fact-generator
```

### 2. Set up the Anthropic API key

This project calls the [Anthropic API](https://www.anthropic.com/) directly from the browser. To use it:

1. Get your API key from [console.anthropic.com](https://console.anthropic.com/)
2. Open `index.html` and find this line:

```js
headers: { 'Content-Type': 'application/json' },
```

3. Add your API key header:

```js
headers: {
  'Content-Type': 'application/json',
  'x-api-key': 'YOUR_API_KEY_HERE',
  'anthropic-version': '2023-06-01'
},
```

> ⚠️ **Never commit your API key to a public repo.** For production, route calls through a backend proxy that injects the key server-side.

### 3. Open in browser

```bash
open index.html
# or just double-click the file
```

No server required for local use.

## 🔧 Customization

### Add more topics

In `index.html`, add a new button inside the `#topics` div:

```html
<button class="topic-btn" data-topic="Dinosaurs">🦕 Dinosaurs</button>
```

The `data-topic` value is passed directly to the AI prompt — no other changes needed.

### Change the prompt

Find the `content` field in the `fetch` call and edit the prompt:

```js
content: `Give me ONE surprising fact about "${selectedTopic}". ...`
```

---

## 🛡️ Security Note

Direct browser-to-API calls expose your API key in the client. For any public deployment:

- Use a serverless function (e.g. Vercel, Netlify Functions, Cloudflare Workers) as a proxy
- Store the API key in an environment variable on the server
- Rate-limit requests per user/IP
